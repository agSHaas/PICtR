# Save plot

Wrapper function for saving plots as a pdf file.

## Usage

``` r
save.plot(plot, path = output_plot_path, file, dim_h = 7, dim_w = 5)
```

## Arguments

- plot:

  The plot.

- path:

  Directory path.

- file:

  The file name.

- dim_h:

  Height parameter for [`pdf`](https://rdrr.io/r/grDevices/pdf.html)

- dim_w:

  Width parameter for [`pdf`](https://rdrr.io/r/grDevices/pdf.html)

## Value

None
